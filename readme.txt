Инструкция:
1) Воспользоваться готовым запущенным сайтом: http://51.250.45.241:5000/
    1. Нажать на кнопку выбрать файл
    2. Выбрать видео для обработки на ПК
    3. Нажать на кнопку обработать видео
    4. Дождаться конца обработки
    5. В блоке ниже появяться готовые клипы, их можно просмотреть, загрузить

2) Развернуть у себя на ПК:
    1. скачать файл yolov3 и перенести в папку проекта: https://drive.google.com/file/d/1xGpFnh5IObji5ul1pBoGqeaBWGynSvn0/view?usp=sharing
    2. Чтобы запустить программу с веб-сайтом, нужно запустить файл app.py
    3. После запустить сайт в браузере: http://127.0.0.1:5000/
    4. На сайте можно отправить видео на обработку, скачать полученные клипы
