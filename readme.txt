Инструкция:

1. скачать файл yolov3: https://drive.google.com/file/d/1xGpFnh5IObji5ul1pBoGqeaBWGynSvn0/view?usp=sharing
и перенести в папку проекта
2. Чтобы запустить программу с веб-сайтом, нужно запустить файл app.py
3. После запустить сайт в браузере: http://127.0.0.1:5000/
4. На сайте можно отправить видео на обработку, скачать полученные клипы
